Meus Trabalhos

Ao longo da minha jornada como desenvolvedor web, busquei criar projetos que demonstram meu foco em funcionalidade, design responsivo e interatividade. Aqui estão alguns exemplos:

  1. Quiz Interativo
    Desenvolvi um quiz que inclui perguntas sobre filmes e músicas, implementando lógica de contagem de pontos e validação de respostas corretas. Esse projeto me permitiu aplicar técnicas de JavaScript e melhorar a experiência do usuário com um design envolvente e interativo.
  2. Sistema de Cadastro de Usuários
    Criei um sistema completo para o cadastro de usuários, onde implementei validação de dados e uma interface de fácil uso. Esse projeto incluiu o uso de conceitos de backend para armazenamento e gerenciamento de informações, consolidando conhecimentos em Node.js e Express.
  3. Landing Page Dinâmica
    Desenvolvi uma landing page moderna e responsiva, com layout otimizado para diferentes dispositivos e integração de APIs para exibir conteúdo dinâmico. Nesse projeto, trabalhei com HTML, CSS e JavaScript, focando na usabilidade e no design atrativo.
